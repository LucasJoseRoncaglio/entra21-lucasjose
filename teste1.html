8) A fim de manter o calendário anual ajustado com o movimento de translação da Terra, criou-se os anos bissextos, que têm 366 dias em vez dos 365 presentes nos anos normais. Para determinar se um ano é bissexto, é necessário saber se ele é multiplo de 4. Não pode ser múltiplo de 100, exceto se for também múltiplo de 400. Com isso em mente, desenvolva uma função que recebe um número correspondente a um ano e retorna se ele é bissexto ou não (true ou false). Após isso mostre na tela uma mensagem "O ano é bissexto" ou "O ano não é bissexto". Exemplo: checarAnoBissexto(2020) // true checarAnoBissexto(2100) // false
